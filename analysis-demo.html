实时演示

选择一个场景，或输入你自己的情境：

在你的网页中使用

1. 引入两个文件：

<script src="analysis-engine.js"></script> <script src="analysis-renderer.js"></script>

2. 一行代码完成分析+渲染：

// 最简用法 quickAnalyze('#output', { scenario: '你的情境描述', context: { role: '职场新人', stakes: '中等' }, options: [ { id: 'a', text: '当面质问', tags: ['direct', 'confrontational'] }, { id: 'b', text: '装作不知道', tags: ['passive', 'avoidant'] }, { id: 'c', text: '找领导谈', tags: ['hierarchical'] }, ] });

3. 高级用法 — 自定义框架组合：

const engine = new AnalysisEngine({ frameworks: ['gameTheory', 'powerDynamics', 'moralDilemma'], depth: 'deep' }); const result = engine.analyze(input); // result 包含： // .chainOfThought — 链式推理步骤 // .frameworks — 各框架的分析结果 // .options — 带评分的选项排序 // .tensions — 核心张力（矛盾点） // .summary — 总结判断

4. 在文字游戏中使用 — 动态分析玩家选择：

// 玩家做出选择后，分析其后果 function onPlayerChoice(choice) { const result = engine.analyze({ scenario: currentScene.description, context: { role: player.identity, stakes: currentScene.stakes, trust_level: player.trustScore, history_length: player.choices.length, }, options: currentScene.options.map(o => ({ id: o.id, text: o.text, tags: o.analysisTags, // 每个选项预设的分析标签 })) }); // 渲染分析结果到场景面板 renderer.render(result); // 或者只取洞察文本，嵌入到叙事中 const topInsight = result.insights .filter(i => i.severity === 'high')[0]; appendNarrative(topInsight.text); }

选项标签参考

给选项打标签，引擎会根据标签在各维度上评分：

权力: direct, confrontational, passive, avoidant, coalition, alliance, information, leverage, hierarchical, escalation
群体: public, visible, conformist, go_along, whistle, dissent, private, discrete
道德: ethical, principled, pragmatic, practical, selfish, exploit, sacrifice, altruistic, deceptive, manipulative
博弈: cooperate, collaborate, defect, betray, signal, commit, retaliate, punish, negotiate, mediate
信息: information, research, reveal, transparency, conceal, secret, demonstrate
反操控: boundary, assertive, seek_help, support, comply, submit, manipulate_back, document, evidence
交换: reciprocate, return_favor, invest, build_trust, withdraw, exit, clarify
认知: evidence, data, reflect, pause, impulsive, emotional, perspective, reframe, consult, outside_view
